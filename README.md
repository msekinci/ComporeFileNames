# ComporeFileNames
This project is generally used to find the missing ones by comparing the file names in the folder with the list of names in the file.

In the content of the project;
- Extract data from Excel and convert to List,
- Picking the file names in the folder and converting them to List,
- Method of converting the entered string value to uppercase letters,
- Method to delete unwanted parts from the entered string value (to remove extensions from file names)
